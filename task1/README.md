# Домашнее задание №1  
## CLI-инструмент для получения информации о репозитории GitHub

---

## Запуск

Требуется Go 1.25+.

Склонируйте репозиторий и выполните:

```bash
go run .
```

После запуска программа попросит ввести репозиторий:

```bash
GitHub Repository Info CLI
Введите имя репозитория в формате owner/repo или два аргумента owner repo
> golang/go
```

Примеры:

```bash
> golang/go
> golang go
> torvalds/linux
```

### Что выводит программа

- Имя репозитория
- Описание
- Количество звёзд
- Количество форков
- Дата создания
